## 🔎 Troubleshooting common issues

**The program does not find my save files:**
Check that No Man's Sky is installed correctly. Verify that you have played the game at least once to generate the initial save data.

**The program window is blank or crashes:**
Ensure you have the latest Microsoft Visual C++ runtime components installed on your computer. Visit the official Microsoft website to download the latest redistributable package.

**My changes did not appear in the game:**
Restart No Man's Sky completely. The game sometimes caches save information in its memory. A full restart forces the game to read the updated file from your hard drive.
